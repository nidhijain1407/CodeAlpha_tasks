# 2. Stock Trading Platform

A console-based Java application that simulates stock trading operations.

## Features

- View Available Stocks
- Buy Stocks
- Sell Stocks
- View Portfolio
- Transaction History

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## Classes Used

### Stock
Stores stock details:
- Symbol
- Name
- Price

### User
Manages:
- Portfolio
- Transactions

### Transaction
Stores:
- Buy/Sell Type
- Stock Name
- Quantity
- Amount
